# ✅ Mood & Weather-Based To-Do App

A smart, dynamic to-do application that helps you manage your tasks based on your **mood** and **local weather conditions**. Built for productivity, personalization, and ease of use — with real-time suggestions powered by an external weather API.

---

## 🌟 Overview

This is a feature-rich to-do app designed as part of a developer bootcamp. Users can:
- Create, edit, complete, and delete tasks
- Set their mood and receive personalized suggestions
- Get weather-based task recommendations
- Log in to access their own task list

This app combines frontend design, interactive UI logic, and API integration — making it a great full-stack learning project.

---

## 🔧 Features

- 📝 Add, edit, complete, and delete tasks
- 🎭 Mood selector that influences suggestions
- 🌦️ Live weather integration using an external Weather API
- 🔐 User login for personalized data
- 🎨 Clean, responsive UI (based on a Figma prototype)
